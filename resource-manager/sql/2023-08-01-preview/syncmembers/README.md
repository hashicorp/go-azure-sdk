
## `github.com/hashicorp/go-azure-sdk/resource-manager/sql/2023-08-01-preview/syncmembers` Documentation

The `syncmembers` SDK allows for interaction with Azure Resource Manager `sql` (API Version `2023-08-01-preview`).

This readme covers example usages, but further information on [using this SDK can be found in the project root](https://github.com/hashicorp/go-azure-sdk/tree/main/docs).

### Import Path

```go
import "github.com/hashicorp/go-azure-sdk/resource-manager/sql/2023-08-01-preview/syncmembers"
```


### Client Initialization

```go
client := syncmembers.NewSyncMembersClientWithBaseURI("https://management.azure.com")
client.Client.Authorizer = authorizer
```


### Example Usage: `SyncMembersClient.CreateOrUpdate`

```go
ctx := context.TODO()
id := syncmembers.NewSyncMemberID("12345678-1234-9876-4563-123456789012", "example-resource-group", "serverValue", "databaseValue", "syncGroupValue", "syncMemberValue")

payload := syncmembers.SyncMember{
	// ...
}


if err := client.CreateOrUpdateThenPoll(ctx, id, payload); err != nil {
	// handle the error
}
```
