
## `github.com/hashicorp/go-azure-sdk/resource-manager/web/2023-01-01/resourcehealthmetadata` Documentation

The `resourcehealthmetadata` SDK allows for interaction with Azure Resource Manager `web` (API Version `2023-01-01`).

This readme covers example usages, but further information on [using this SDK can be found in the project root](https://github.com/hashicorp/go-azure-sdk/tree/main/docs).

### Import Path

```go
import "github.com/hashicorp/go-azure-helpers/resourcemanager/commonids"
import "github.com/hashicorp/go-azure-sdk/resource-manager/web/2023-01-01/resourcehealthmetadata"
```


### Client Initialization

```go
client := resourcehealthmetadata.NewResourceHealthMetadataClientWithBaseURI("https://management.azure.com")
client.Client.Authorizer = authorizer
```


### Example Usage: `ResourceHealthMetadataClient.GetBySite`

```go
ctx := context.TODO()

read, err := client.GetBySite(ctx, id)
```
