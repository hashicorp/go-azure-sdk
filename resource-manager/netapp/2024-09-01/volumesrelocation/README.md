
## `github.com/hashicorp/go-azure-sdk/resource-manager/netapp/2024-09-01/volumesrelocation` Documentation

The `volumesrelocation` SDK allows for interaction with Azure Resource Manager `netapp` (API Version `2024-09-01`).

This readme covers example usages, but further information on [using this SDK can be found in the project root](https://github.com/hashicorp/go-azure-sdk/tree/main/docs).

### Import Path

```go
import "github.com/hashicorp/go-azure-sdk/resource-manager/netapp/2024-09-01/volumesrelocation"
```


### Client Initialization

```go
client := volumesrelocation.NewVolumesRelocationClientWithBaseURI("https://management.azure.com")
client.Client.Authorizer = authorizer
```
