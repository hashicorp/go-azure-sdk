
## `github.com/hashicorp/go-azure-sdk/resource-manager/automation/2019-06-01/dsccompilationjob` Documentation

The `dsccompilationjob` SDK allows for interaction with Azure Resource Manager `automation` (API Version `2019-06-01`).

This readme covers example usages, but further information on [using this SDK can be found in the project root](https://github.com/hashicorp/go-azure-sdk/tree/main/docs).

